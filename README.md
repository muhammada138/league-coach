# Rift IQ

AI-powered League of Legends coaching. Enter your Riot ID, get a breakdown of your last 5 ranked games, and receive targeted tips generated by an LLM — grounded in your actual stats vs. lobby averages.

## What it does

Most coaching tools give you raw stats and leave the interpretation to you. Rift IQ instead compares your performance **against the other 9 players in each of your games**, identifies where you're falling behind, and generates **3–5 specific, actionable coaching tips** ranked by impact.

**Key features:**

- **Ranked game analysis** — pulls your last 5 solo/duo games via the Riot API
- **Lobby-relative stats** — compares your KDA, CS/min, vision score, damage, gold, and more against per-game averages (not global averages)
- **AI coaching tips** — LLaMA 3.3 70B (via Groq) generates structured tips grouped by Vision, Combat, and Economy — no generic advice
- **LP trend graph** — estimates LP gain/loss across your last 5 games
- **Expandable scoreboards** — full 10-player scoreboard with performance scores (0–100) for every game
- **Dark/light mode** — League-themed UI with glassmorphism cards and smooth animations

## Getting started

You'll need a [Riot API key](https://developer.riotgames.com/) and a [Groq API key](https://console.groq.com/).

### Backend

```bash
cd backend
pip install -r requirements.txt
cp .env.example .env   # add your API keys
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Open `http://localhost:5173`, enter a Riot ID (e.g. `Faker#KR1`), and hit Analyze.

```env
RIOT_API_KEY=your_riot_api_key
GROQ_API_KEY=your_groq_api_key
RIOT_REGION=na1          # your server region (na1, euw1, kr, etc.)
RIOT_ROUTING=americas    # routing region (americas, europe, asia)
```

## API endpoints

| Method | Path | Description |
|---|---|---|
| `GET` | `/summoner/{game_name}/{tag_line}` | Resolve Riot ID to PUUID |
| `GET` | `/profile/{puuid}` | Rank, LP, W/L, summoner level |
| `GET` | `/analyze/{puuid}?game_name={name}` | Full analysis with AI coaching tips |
| `GET` | `/match/{match_id}/scoreboard` | 10-player scoreboard for a match |

---

## How the coaching works

The `/analyze` endpoint:

1. Fetches your last 5 ranked match IDs
2. Pulls full match data for each game in parallel
3. Extracts your stats and the lobby average across 9 key metrics
4. Builds a structured prompt with per-game deltas (you vs. average)
5. Calls Groq's LLaMA 3.3 70B with a strict system prompt that enforces actionable, grouped tips
6. Returns everything — stats, game history, coaching narrative — in one response

The AI is explicitly instructed not to give generic advice. Tips reference your actual numbers (e.g. "your vision score was 4 vs. a lobby average of 18").
